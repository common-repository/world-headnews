== Description ==

Display actual news that recently happens on your dashboard page. It also create an animated headline on every admin page. So you can keep up the story up to date.

The story are fetched from blogsearch,google.com so you get the current fresh and hot news on the net!


== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
